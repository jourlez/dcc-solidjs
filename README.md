### Clone to local

If you prefer to do it manually

```bash
git clone https://github.com/jourlez/dcc-solidjs
cd dcc-solidjs
```

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install --shamefully-hoist
```

Start the development server:

```bash
npm run dev
```
